DESCRIPTION
	Folder of the application built in Java (using NetBeans) for the Advanced Programming project of the bachelor's degree in computer engineering at the University of Pisa.
	This application allows multiple users to record their running sessions, calculate the calories and fat burned and display their training sessions over time in a graph.
	This examination and its documentation are in Italian (language used in the bachelor's degree).

The folder contains:
	This files:
	- db.sql: contains the SQL DB; 
	
	This document:
	- Analisi.pdf: contains the actions performed by the user and the system in an error-free use case. It explains the contents of the local configuration xml file, 
		       what input is cached locally, what data is stored and the format of remote logs.
	- Documento di progetto.pdf: class diagram in UML with an explanation of each class.
	- Manuale utente.pdf: contains the user manual with views of the application in use and an explanation.

	This folder:
	- Faster: containing all files concerning the application code;
